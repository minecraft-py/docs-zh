# 游戏的中文文档
欢迎阅读Minecraft-in-python的官方中文文档。

这个文档主要讲述的是游戏的基本内容，而非特性，那是[Minecraft wiki](https://minecraft.fandom.com/zh/wiki/Minecraft_Wiki)干的事。当然，也会涉及一些实现细节。

中文文档被分成了如下4个部分：

- **开始之前**---您在游玩之前要做什么
- **游玩之时**---准备游玩时，可以读一下这部分
- **致开发者**---讲述游戏特定的实现细节
- **做出贡献**---对于想要为游戏或文档做出贡献的人，应该阅读这一部分

我们欢迎您为该文档做出贡献，或者把它翻译到另一门语言。

那么，开始阅读吧！
